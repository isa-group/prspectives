About PRspectives

PRspectives is a multi-perspective process model editor that has been developed by ISA Research Group. It integrates several technologies that have been developed as part of Research Projects ISABEL, SETI, THEOS and TAPAS. Specifically, it uses:
  • PPINOT to model and analyse process performance indicators
  • CRISTAL to model and analyse the organisational perspective of the process
In addition, it also integrates a model editor based on Oryx Editor.
